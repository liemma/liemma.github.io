Pac-Man on FPGA

Recreated the classic Pac-Man game on a DE1-SoC FPGA using SystemVerilog for VGA rendering and C for game logic. Developed hardware for tile-based graphics, sprite rendering, and PWM-based audio. Integrated memory-mapped I/O for real-time software-hardware communication and sprite updates.

Built a full-stack web app for cataloging songs with search by title, writer, and artist. Created a Flask backend for song management and a jQuery-powered frontend for real-time updates and interactive search functionality.

Maze Joystick Controlled Game

Developed an interactive labyrinth game controlled by an Arduino-powered joystick. Constructed a physical maze using cardboard walls on a plywood base and integrated servo motors to tilt it in two planes based on joystick input.
